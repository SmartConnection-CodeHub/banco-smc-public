🚨 PROBLEMA PRINCIPAL: La maqueta es solo mobile

VOY v1 es una app desktop-first con sidebar, topbar, modales, paneles laterales deslizantes, y layout de 2 columnas. La maqueta de Celia solo muestra phones de 375px. Falta toda la versión desktop.

Layout actual de VOY v1 (desktop)

  • Topbar: logo VOY + rol badge + buscador central + toggle disponibilidad + notif bell + chat icon + avatar con nombre + logout
  • Sidebar 260px: navegación con iconos FontAwesome + badges de conteo + secciones "Gestión" y "Mi cuenta" + footer con logout
  • Main area: vistas que se muestran/ocultan con showView() — NO es SPA con rutas, es DOM toggling
  • Modales: booking, rating, cotización, chat, notificaciones — overlays con animación
  • Panel lateral deslizante: worker-detail (420px, translateX) con header + body scrolleable + footer con botones
  • Chat panel: panel fijo con header, messages y input
  • Toast container: notificaciones toast animadas
