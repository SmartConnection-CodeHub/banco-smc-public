🤖 Guía de Instalación

VS Code + Claude Code

Conecta tu editor favorito con el CLI de IA más poderoso. Trabaja con Claude directamente desde tu proyecto.

📦 Requisitos previos
🟢
Node.js versión 18 o superior
💙
VS Code cualquier versión reciente
🔑
Cuenta Anthropic claude.ai o API key
💻
Terminal macOS / Linux / Windows WSL
1
Verificar Node.js
CLI

Abre la terminal integrada de VS Code y verifica que tengas Node.js 18+ instalado.

$ node --version
# debe mostrar v18.x.x o superior

$ npm --version
# debe mostrar 8.x.x o superior
💡 ¿No tienes Node.js? Descárgalo en nodejs.org o usa brew install node en macOS.
2
Instalar Claude Code CLI
CLI

Instala el paquete global de Claude Code con npm. Este es el motor que alimenta todo.

$ npm install -g @anthropic-ai/claude-code
# verificar instalación
$ claude --version
3
Abrir la terminal integrada en VS Code
VS Code
⚠️ Ojo: En el Marketplace de VS Code aparecen muchas extensiones llamadas "Claude Code" pero ninguna es oficial de Anthropic. Son wrappers de terceros. No instales ninguna — la integración real es vía terminal.

Claude Code se integra con VS Code a través de la terminal integrada. Sin extensiones, sin configuración extra.

💻 Terminal integrada método oficial
  • Abre tu proyecto en VS Code
  • Presiona ⌘ + J para abrir la terminal
  • Ejecuta claude y ya está
  • Lee el contexto del proyecto automáticamente
  • Acceso a todos los archivos del workspace
🖥️ Terminal externa
  • Abre Terminal.app o iTerm2
  • Navega con cd ~/tu-proyecto
  • Ejecuta code . para abrir VS Code
  • Luego ejecuta claude
  • Útil si prefieres terminal separada
1
Con VS Code abierto, presiona ⌘ + J
2
Se abre el panel inferior → selecciona pestaña Terminal
3
Escribe claude y presiona Enter
4
Claude detecta el proyecto automáticamente y está listo
4
Autenticar con Anthropic
Auth

En la terminal de VS Code, ejecuta Claude por primera vez. Te pedirá autenticación.

# Navega a tu proyecto primero
$ cd ~/tu-proyecto

# Inicia Claude Code
$ claude
Opción A — Claude.ai

Si tienes plan Claude Pro/Max, autentícate con tu cuenta claude.ai. Abrirá el navegador automáticamente.

Opción B — API Key

Usa tu API key de Anthropic. Pégala cuando la solicite. Se guarda en ~/.claude.

5
Abrir tu proyecto y ejecutar
Listo 🎉

Con el proyecto abierto en VS Code, Claude Code lee automáticamente el contexto de tus archivos.

# Abrir VS Code en el directorio actual
$ code .

# Desde la terminal integrada de VS Code:
$ claude

# O pide algo directo:
$ claude "explica qué hace este proyecto"
⚡ Claude Code lee tu package.json, estructura de archivos y código automáticamente. Usa claude --help para ver todos los comandos disponibles.
