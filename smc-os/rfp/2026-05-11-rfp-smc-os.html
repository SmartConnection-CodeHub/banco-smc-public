01🎯 Contexto

Guillermo opera Smart Connection en formato consultora boutique · clientes B2B (Cencosud · Parque · MeLi) + proyectos propios (Kanki · InfoPet · Marketplace · VOY) + canales business-development. Tradicionalmente esto requiere 20+ personas (PM · BA · Architect · devs · QA · marketing · contabilidad · etc).

💡 Insight: En vez de contratar 23 personas (costo ~$30M CLP/mes salario + cargas) · armamos 23 agentes Claude Code especializados que actúan como equipo virtual. Costo: plan Claude Max ~$200/mes. Diferencia: $30M vs $200 = 150x menos costo · mismo output con asistencia humana selectiva.

02🔥 Problema

  • Context loss · cada sesión Claude empezaba desde cero · re-explicar todo
  • Inconsistencia · sin agentes especializados · respuestas generales sin profundidad SMC
  • Sin orquestación · usuario debía decidir manualmente qué agente invocar
  • Sin gobernanza · sin reglas técnicas estables · cada sesión re-inventaba
  • No escala con Sebas · primer empleado necesita onboarding automático

03💡 Hipótesis

Hipótesis: 23 agentes especializados con loops formalizados (Orchestrator · Conversacional · ReAct · Pipeline · Reflexion · Background · Híbrido) + memoria persistente + Cerebro orquestador = equipo virtual escalable a costo fijo bajo.

04🛠️ Solución

Arquitectura

  • 23 agentes · .md files en ~/.claude/agents/ · cada uno con identidad · rol · skills
  • 7 loops formalizados · Orchestrator (PM) · Conversacional (BA · Architect) · ReAct (Hoku · Backend · Frontend) · Pipeline (User-QAS · BD · etc) · Reflexion (Cerebro) · Background (Security centinela) · Híbrido
  • 3 modelos Claude · Opus 4.7 (7 agentes complejos) · Sonnet 4.6 (9 agentes) · Haiku (1 agente DevOps)
  • Memorias persistentes · ~/.claude/projects/.../memory/ · ~80 archivos · MEMORY.md como index
  • Skills auto-load · triggers por keywords/proyecto/agente · contextual
  • Hooks · SessionStart · PreToolUse guardrails · Stop notification · session_end_report
  • 17 connectors · LLMs (Claude · Groq · Gemini · etc) + MCPs (Context7 · GitHub · Firecrawl · etc)
