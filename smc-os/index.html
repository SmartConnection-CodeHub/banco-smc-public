Tabla #3🔄 Orquestación · 5 capacidades automáticas Cerebro
Cómo Cerebro decide solo cuándo invocar agentes · sin pedir OK a Guillermo.
1
Handoff
Agente A detecta necesidad de B
Cerebro mueve la conversación a B sin preguntar. Visible: "🐕 BA entra · valida flujo"
2
Paralelismo
Tareas independientes detectadas
Activa Backend + Frontend simultáneamente · no espera secuencial
3
Anti-loop
Detecta A→B→C→A
Intercepta · resume · pide dirección. Max 3 handoffs sin OK del usuario
4
Streaming vivo
Tareas en cola con AFK usuario
Ejecuta mientras usuario piensa. Próxima respuesta muestra progreso real
5
Selección agente
Usuario habla sin invocar agente
Elige el más adecuado al contexto sin preguntar (salvo ambigüedad)

Tabla #4🎚️ 4 niveles de autonomía
Cuándo Cerebro ejecuta solo · cuándo pregunta · regla de las 3 confirmaciones.
🟢 NIVEL 0
Auto siempre
Cero riesgo · sin avisar
Read · grep · curl · crear memoria · sync repos
🟢 NIVEL 1
Auto + notify
Ejecuta · avisa 1 línea
Edit en plan acordado · push commits · renames consistentes
🟡 NIVEL 2
Pregunta 1ª vez
Después auto en sesión
Crear agente · repo nuevo · infra cloud (S3/CF/RDS)
🔴 NIVEL 3
Pregunta siempre
Sin importar confirmaciones previas
Push a prod · delete repo · rm -rf · email cliente · cambios legales
