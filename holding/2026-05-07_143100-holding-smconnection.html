Macro · proceso end-to-end del holding

El flujo unifica los 4 escenarios típicos del año contable:

🟢

A · Operación normal

Cliente paga → SpA recibe → IVA débito → SpA gasta → IVA crédito → F29 mensual día 12.

📦

B · Importación directa

Kanki o Marketplace importan directo → DUS aduana → IVA aduana crédito → llega stock → vende al cliente final → ciclo IVA continúa. Si conviene: contratan a Nextport como servicio externo.

🎯

C · Fondos públicos

Postula → ejecuta plan al peso → rinde con factura+medio pago+foto → desembolso 5-6 meses.

🏛️

D · ChileCompra

Inscripción ChileProveedores → licita → garantía bancaria → Estado paga 60-90 días → factoring opcional.
