🎯 Mecánica

TURNO INICIA ↓ 🧠 CEREBRO escanea contexto: 1. Último mensaje del usuario → busca keywords 2. CWD actual (proyecto) → match project 3. Agente activo → match agentes destino 4. Archivos abiertos/citados → match files ↓ Por cada skill con match: → carga skill como contexto extra → muestra micro-badge: "⚡ /sap-integration cargado" ↓ Agente responde CON skills ya disponibles

📋 Frontmatter de skill con triggers

--- name: sap-integration description: SAP CPI · iFlows · mapeos auto_load: true triggers: keywords: [sap cpi, cpi, iflow, integration suite, sap pi] projects: [duoc, cencosud, meli-valmer, parque-arauco] agents: [architect, hoku-dev, backend] files: ["*.iflow", "*.groovy", "*.xsd"] priority: high ---
