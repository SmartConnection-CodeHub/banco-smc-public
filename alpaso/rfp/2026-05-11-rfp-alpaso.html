01🎯 Contexto

Estudio de danza familiar en Concón · pequeño · necesita presencia web elegante para mostrar clases · profesoras · galería de eventos. NO necesita reservas online ni ecommerce. Cliente sensible al precio (cuota mensual baja del estudio).

💡 Insight: Para landings sin lógica de negocio · el stack más barato gana siempre. HTML estático + S3 + CloudFront = $0.50/mes vs $30/mes Shopify/WordPress. Cliente paga 1 vez el desarrollo y el sitio vive años sin costo recurrente.

02🔥 Problema

  • Estudio sin presencia web · clientes preguntan vía WhatsApp + Instagram

  • Presupuesto chico · no puede pagar Shopify/WordPress mensual

03💡 Hipótesis

Hipótesis: HTML puro + S3 + CloudFront = sitio elegante · responsive · <$0.50/mes infra · maqueta = producción (mismo HTML) · patrón replicable a 10+ clientes pequeños tipo Alpaso.
