🚀 4 Pilares producto · diferenciación

🤖
1. Tsukai IA · recomendaciones
Qué: Chat IA recomienda por raza · edad · condición · alergias
Diferenciación: 95% ecommerce pet Chile no tiene asistente personalizado
Impact: +25-40% carrito promedio (benchmark Chewy)
Stack: Groq Llama 3.3 → Scout → Gemini → DeepSeek
📦
2. Sync stock real-time
Qué: Bsale ↔ Jumpseller bidireccional · stock 0 marca agotado en cascada
Diferenciación: elimina 10-15% revenue perdido por desync
Impact: +$2.5M-4.5M CLP/año recuperado (tienda $30M revenue)
Stack: Supabase Realtime · webhooks bidireccionales
🏄
3. InfoPet Reñaca panel local
Qué: Panel local "Mis Olas" para tienda física Reñaca · ranking ventas + wallpaper
Diferenciación: conecta digital con experiencia física · cultura surf local
Impact: engagement empleados +60% · gamificación ventas
Stack: Next.js panel dedicado · live data Supabase
🧾
4. OCR Facturas Mobile
Qué: Mobile sube foto factura proveedor · OCR + autocarga Bsale
Diferenciación: elimina carga manual 10 min → 30 segundos
Impact: 8-10h/semana ahorradas operario carga proveedores
Stack: Cloud OCR (Google Vision o AWS Textract) + Bsale API
