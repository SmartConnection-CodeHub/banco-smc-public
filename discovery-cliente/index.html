Tabla #1🎯 SPIN Selling · 4 tipos de preguntas
Framework para descubrir el problema real del cliente · NO el síntoma.
S
Situación
¿Cómo está hoy? Stack actual · equipo · procesos
P
Problema
¿Qué le frustra? Dolores · gaps · ineficiencias
I
Implicación
¿Qué le cuesta no resolverlo? Tiempo · dinero · oportunidad
N
Necesidad-beneficio
¿Cómo sería si lo resolviera? Visualizar el "después"

Tabla #3🛣️ Pipeline 3 fases
Roadmap del Discovery · de manual a app web · cada fase reduce tiempo del operario.
1
FASE 1 · Manual con plantillas
Plantillas HTML editables · Business-Dev copia transcript y pega · genera infografía + reporte. Tiempo: 30 min por discovery.
2
FASE 2 · Script bash + Claude
Script lee transcript .txt · llama Claude API con prompt SPIN/BANT · genera HTMLs automáticos. Tiempo: 5 min por discovery.
3
FASE 3 · App web self-service
Cliente sube transcript desde intranet.smconnection.cl · genera todo solo · descarga PDF + envío email. Tiempo: 1 min cliente · cero operario.
