👤 Tú
El harness ejecuta un shell script en momentos clave (inicio, fin, bloqueo).
No consume tokens. No necesita que Claude "recuerde" hacerlo.

Stop
Voz al terminar
say "Tarea completada"
Cuando Claude termina una tarea, macOS habla en voz alta. No necesitas mirar la pantalla ni esperar la notificación visual — lo escuchas aunque tengas auriculares puestos o estés en otra app.

SessionStart + Stop
Mac despierto en deploys
caffeinate -di &
Al iniciar una sesión Claude activa caffeinate, que le dice a macOS "no duermas". Al terminar, lo mata. Sin esto, si tu Mac entra en reposo durante un deploy o build largo, el proceso puede pausarse o morir.
▶️
Hook SessionStart
Inicia caffeinate en background
caffeinate -di & echo $! > /tmp/claude_caffeinate.pid
🍎
macOS — Power Management
Suspende el sleep del sistema y del disco
-d = no sleep del sistema / -i = no sleep inactivo
⏹️
Hook Stop
Mata caffeinate al terminar la sesión
kill $(cat /tmp/claude_caffeinate.pid) 2>/dev/null
🛡️
Lo que percibes tú El deploy termina aunque te hayas ido a tomar un café. Sin caffeinate, el Mac duerme y el proceso queda a medias.

PreToolUse
Alerta sonora de bloqueo
sound name "Sosumi"
Cuando Claude intenta ejecutar algo peligroso (rm -rf, DROP TABLE, escribir en .env) el hook lo bloquea con exit 2. Además de rechazar la acción, suena el clásico "Sosumi" de macOS — el sonido de "no se puede".
⚠️
Claude intenta ejecutar
Bash con rm -rf / o DROP TABLE
O escribir en .env / .pem / .key
⚙️
Hook PreToolUse — Harness
Detecta el comando peligroso → exit 2 → bloquea
osascript -e 'beep' -e 'set volume alert volume 100' osascript -e 'display alert "Bloqueado" message "Comando peligroso detectado"'
🍎
macOS
Suena Sosumi + aparece alerta visual
La acción de Claude fue rechazada — no se ejecutó nada
🔴
Lo que percibes tú Escuchas "Sosumi" y aparece una alerta. Sabes al instante que algo se bloqueó sin tener que revisar el terminal.
📋 Configuración en ~/.claude/settings.json
// Agregar dentro de "hooks": {}

// 1. Voz al terminar
"Stop": [{
  "matcher": "",
  "hooks": [{
    "type": "command",
    "command": "say 'Tarea completada'",
    "async": true
  }]
}]

// 2. Caffeinate (agregar a session_audit.sh)
caffeinate -di & echo $! > /tmp/claude_caffeinate.pid
// En Stop → kill $(cat /tmp/claude_caffeinate.pid)

// 3. Sosumi en bloqueos (dentro del PreToolUse Bash existente)
osascript -e 'beep' 2>/dev/null
echo 'BLOQUEADO: comando peligroso' >&2
exit 2
